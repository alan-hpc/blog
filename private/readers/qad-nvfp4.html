NVFP4 推理精度恢复的量化感知蒸馏

Quantization-Aware Distillation for NVFP4 Inference Accuracy Recovery

原文来源 Quantization-Aware Distillation for NVFP4 Inference Accuracy Recovery · NVIDIA · arXiv 2025
作者：Meng Xin, Sweta Priyadarshi, Jingyu Xin, Bilal Kartal et al.

摘要

本技术报告介绍了量化感知蒸馏（Quantization-Aware Distillation，QAD）及其在 NVFP4 量化大语言模型（LLM）和视觉语言模型（VLM）精度恢复方面的最佳实践。QAD 使用 KL 散度损失，将全精度教师模型的知识蒸馏到量化学生模型中。尽管将蒸馏应用于量化模型并非新思路，但我们在今天的 LLM 实践中观察到 QAD 的两个关键优势：

  1. 出色的有效性与稳定性：对于经历多阶段训练后流程（SFT、RL、模型合并）的模型，QAD 表现出卓越的效果和稳定性，而传统 QAT 在这些场景中面临工程复杂性和训练不稳定问题。
  2. 对数据质量的鲁棒性：QAD 对数据质量和覆盖范围具有鲁棒性，无需完整训练数据即可实现精度恢复。

我们在多个训练后模型上评估了 QAD，包括 AceReason Nemotron、Nemotron 3 Nano、Nemotron Nano V2、Nemotron Nano V2 VL（VLM）以及 Llama Nemotron Super v1，结果显示 QAD 能够一致地将精度恢复至接近 BF16 基线的水平。

NVFP4 检查点：Nemotron 3 Nano 30B-A3B、Nemotron Nano 9B V2、Nemotron Nano 12B V2 VL、Llama 3.1 Nemotron Nano VL 8B

QAD 代码：Megatron-LM 版本、NeMo 版本、HuggingFace Transformers 版本

1. 引言

大语言模型的快速扩张推动了对更高效数值格式的需求，以降低训练和推理过程中的计算成本、内存需求和能耗。8 位浮点格式（FP8 和 MXFP8）已成为 LLM 加速训练的主流数据类型。最近，窄精度硬件的进步使 4 位浮点（FP4）成为下一个逻辑演进方向，与 FP8 相比，FP4 可实现两到三倍的算术吞吐量提升，并将内存使用减少约一半。

NVFP4 采用更小的块尺寸（16 vs MXFP4 的 32）、FP8 尺度因子（E4M3）用于细粒度缩放，以及第二级 FP32 缩放以扩大动态范围，在许多模型上展现出优于 INT4 和 MXFP4 的精度。对于超大型 LLM，使用 PTQ 的 NVFP4 在各基准上表现良好；然而，对于小型 LLM，PTQ 带来的精度下降往往不可忽视。

量化感知训练（QAT）是一种有效的基于训练的推理精度恢复方法，自 CNN 时代以来便展现出优异的成果。然而对现代 LLM 而言，QAT 在实践中面临以下重大挑战：

  1. 复杂的训练流程：现代 LLM 经历多阶段训练后处理（SFT、RL、模型合并），难以复现原始训练流程。
  2. 数据可用性和质量：开放模型的原始训练数据可能不可用，公开数据集质量通常较低。

本技术报告评估了用于 NVFP4 推理精度恢复的 量化感知蒸馏（QAD）。QAD 使用原始全精度模型作为教师，通过 KL 散度损失训练量化模型作为学生，而非使用任务特定目标函数。主要发现：

  1. QAD 比 QAT 更好地将量化模型与高精度基线对齐。
  2. QAD 对包含 SFT 和 RL 的多阶段训练后流程模型有效，展现出卓越的稳定性。
  3. QAD 对不完整数据覆盖具有鲁棒性，即使使用部分领域数据也能恢复精度，实现跨领域知识迁移。

2. 背景与相关工作

2.1 NVFP4 格式与训练后量化

NVFP4 格式。NVFP4 是一种 4 位浮点格式，与 FP8 相比提供 2–3× 更高的算术吞吐量，并将内存减少约 1.8×。NVFP4 在 MXFP4 格式基础上扩展：块尺寸从 32 缩减至 16，引入两级缩放（每块 E4M3 尺度加每张量 FP32 尺度）。更小的块尺寸能更局部地适应数据分布，E4M3 尺度提供非 2 的幂次缩放因子以降低量化误差，第二级 FP32 尺度扩展整体动态范围。

训练后量化（PTQ）。PTQ 是一种无需训练的简单低成本方法，通过校准（calibration）过程使用少量代表性数据集确定量化参数（尺度因子和零点）。PTQ 将训练与推理完全解耦，对无法访问原始训练流程或计算资源有限的实践者极具吸引力。PTQ 对 FP8 的大多数 LLM 效果很好，对 NVFP4 在超大型模型上同样良好，但对小型模型和敏感任务往往表现欠佳。

2.2 量化感知训练（QAT）

QAT 仅对权重和激活值进行量化（仅优化前向传播），梯度保持高精度以确保稳定收敛。QAT 使用与原始全精度模型相同的损失函数和训练数据对量化模型进行微调。对于 RL 阶段，等效方法是量化感知强化学习（QARL）——在 actor 的前向传播和 rollout 生成中均应用量化。然而 QARL 的探索较少，现有量化 RL 工作主要聚焦于加速训练吞吐量，而非事后推理精度恢复，这促使我们研究 QAD 作为替代方案。

2.3 用于量化的知识蒸馏

知识蒸馏（KD）通过让学生模型学习教师模型的软标签（概率分布，通常使用 KL 散度损失）来迁移知识。理论研究表明，软标签能以更低方差提供对真实类别概率的更好估计，而匹配教师分布提供了隐式正则化，加速收敛。KD 与量化的结合已在 CNN 和 LLM 领域得到广泛研究。对于 LLM，Liu 等人证明无数据蒸馏极为有效；Kim 等人提出 token 级别缩放的 logit 蒸馏（TSLD）；Du 等人引入结合非对称量化与自蒸馏的 BitDistiller。

现有 QAD 工作主要针对整数量化。本报告证明，简单 KL 散度蒸馏对 NVFP4 推理精度恢复极为有效，尤其对于经历复杂多阶段训练后流程的 LLM——在这些场景中，复现原始训练不切实际，标准 QAT 存在降低模型能力的风险。

3. 量化感知蒸馏（QAD）

3.1 方法概述

QAD 使用原始高精度模型作为教师，通过蒸馏损失训练量化模型作为学生。与标准 QAT 的区别在于损失函数，如图 1 所示：

  • QAT：使用与原始模型训练相同的任务特定损失（如语言建模的下一个 token 交叉熵）。
  • QAD：使用高精度教师模型与量化学生模型之间的 KL 散度。

形式上，对于给定输入 $x$ 和词表 $V$，QAD 损失为教师与学生分布之间的 KL 散度：

$$\mathcal{L}_{\text{QAD}} = D_{\text{KL}}(p_{\text{teacher}} \| p_{\text{student}}) = \sum_{y \in V} p_{\text{teacher}}(y|x) \log \frac{p_{\text{teacher}}(y|x)}{p_{\text{student}}(y|x)}$$

3.2 用于训练后模型的 QAD

对于单阶段训练后模型，QAT 可以达到与 QAD 相当的性能（见附录 A）。然而，现代最先进的 LLM 通常经历涉及 SFT、RL 和模型合并的复杂多阶段流程。在这种情况下，更实际的方法是以单阶段运行 QAD 或 QAT（使用 SFT 数据混合或 RL 提示的模型生成数据）。在此简化设置中，QAD 始终优于 QAT，无论原始训练流程多么复杂，都能达到接近 BF16 的精度。

SFT 密集型模型。我们在 Llama Nemotron Super V1 49B 和 Nemotron Nano 9B V2 两个多阶段训练后模型上评估了 QAD。表 2 显示，QAD 在两个模型的所有挑战性推理基准上均优于 QAT——对于 Llama Nemotron Super V1，QAD 在 AIME25（+4.1%）和 GPQA-D（+1.2%）上尤为突出；对于 Nemotron Nano 9B V2，QAD 在 AIME25（+4.4%）和 GPQA-D（+5.8%）上显著超越 QAT。

3.3 对不完整域覆盖的鲁棒性

3.4 训练与评估设置

量化配置。Llama Nemotron Super V1 和 AceReason Nemotron 所有 GEMM 层量化为 NVFP4；Nemotron Nano 9B V2（混合 Mamba-Transformer）采用选择性量化，保持注意力层和首尾各两层为 BF16；Nemotron 3 Nano（MoE 混合 Mamba-Transformer，仅 6 个自注意力层）保持 6 个自注意力层及其前置 Mamba-2 层为 BF16，其余量化为 NVFP4，KV-Cache 量化为 FP8。

超参数。使用保守学习率（1e-6 或 1e-5），softmax 温度 $T=1$。各模型收敛所需 token 量：Llama Nemotron Super V1 49B 约 3 亿；Nemotron Nano 9B V2 约 60 亿；Nemotron Nano 12B V2 VL 约 5 亿；Nemotron 3 Nano 30B-A3B 约 25 亿；AceReason Nemotron 7B 约 8 亿。这些数据量均显著少于原始训练后处理所需的数据量。

4.2 学习率

总体建议学习率在 1e-5 到 1e-6 之间。对于 SFT 训练模型，使用不超过原始 SFT 学习率的值效果最佳（Nemotron Nano 9B V2 最优 1e-6；Nemotron Nano 12B v2 VL 最优 2e-6，比原始 SFT 学习率 2e-5 低 10 倍）。对于 RL 训练模型，由于最终 RL 阶段使模型偏离了冷启动 SFT 数据分布，QAD 可从更大学习率中受益（AceReason Nemotron 最优 1e-5，远大于典型 RL 学习率约 1e-6）。

5. 结论

本技术报告提出 QAD 作为恢复 NVFP4 格式量化 LLM 和 VLM 推理精度的实用有效方法。通过在 Nemotron 系列、Llama Nemotron Super 和 AceReason Nemotron 上的实验，我们证明 QAD 在广泛任务上可靠地将 NVFP4 模型精度恢复至接近 BF16，涵盖经过复杂 SFT、RL 和模型合并流程的模型——在这些场景中，复现原始训练流程对标准 QAT 不切实际。

消融研究进一步证明，QAD 对数据覆盖范围和质量具有鲁棒性：可利用部分领域或合成数据，即使在随机 token 上训练也保持稳定。结合相对较低的数据和计算需求，这些特性使 QAD 成为 PTQ 不足时 NVFP4 精度恢复的实用默认方案。

D. QAT/QAD 与原生量化训练的比较

原生量化训练的目标是降低训练成本，对三个 GEMM（Fprop、Wgrad、Dgrad）均进行量化，典型代表包括 DeepSeek V3 的 FP8 训练及 NVFP4/MXFP4 预训练。QAT/QAD 则仅对权重和激活值量化，梯度保持高精度——反向传播的两个 GEMM（Wgrad 和 Dgrad）无法以低精度计算，QAT 仅模拟前向传播中的推理精度条件，而非加速训练本身。
